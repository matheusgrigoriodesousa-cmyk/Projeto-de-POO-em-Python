# 📚 Sistema de Cálculo de Média de Alunos em Python

Este projeto foi desenvolvido em Python com o objetivo de calcular a média ponderada de um aluno e determinar automaticamente sua situação acadêmica utilizando Programação Orientada a Objetos (POO).

O sistema utiliza três notas:
- NP1 (peso 4)
- NP2 (peso 4)
- PIM (peso 2)

---

# 🧮 Regra de Cálculo

Média = (NP1 × 4 + NP2 × 4 + PIM × 2) / 10

---

# 📊 Situação do Aluno

- Média ≥ 6 → Aprovado  
- Média ≥ 4 e < 6 → Recuperação  
- Média < 4 → Reprovado
